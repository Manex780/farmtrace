# 🌱 Farm Traceability System

This project is a simple web app that:
- Reads sensor data (weight, temperature, humidity) from Google Sheets
- Displays it in a dashboard
- Generates a QR-based traceability record
- Shows farm location on Google Maps

## How to Run
1. Download project
2. Open index.html in browser
